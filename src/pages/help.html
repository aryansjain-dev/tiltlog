Getting Started

To start using the motion control system:

  1. Open the control website on your phone or laptop.
  2. Make sure your device supports motion sensors.
  3. Allow sensor access if the browser asks for permission.
  4. Press Start to begin tracking motion.

Once started, the page will switch to full-screen mode and the motion values will begin updating.

Interface Overview

The control interface displays the current gyroscope values measured by the device.

These values change as the device moves or tilts, allowing the system to detect motion in real time.

The interface is designed to be simple and distraction-free so that users can focus on controlling the vehicle.

Control Buttons

Start
The Start button begins the motion tracking session.

When pressed:

  • Motion sensors begin recording data
  • The website enters full-screen mode
  • Motion values start updating in real time

Control Buttons

Stop / ESC
Pressing Stop button or the ESC key stops the motion tracking session.

This action:

  • Ends full-screen mode
  • Stops the motion tracking display

Control Buttons

Reset
The Reset button recalibrates the system.

When reset is pressed:

  • All displayed values return to 0
  • The current device position becomes the new reference point
  • Motion values are measured relative to this new position

Tips for Better Control

For the best experience:

  • Hold the device steady before pressing Reset
  • Use smooth movements rather than sudden jerks
  • Keep the device oriented like a steering wheel
  • Ensure motion sensor permission is enabled

Troubleshooting

Motion values are not changing

Possible causes:

  • Sensor permission is not enabled
  • The browser does not support motion sensors
  • The device does not have a gyroscope

    • Try refreshing the page and allowing motion sensor access when prompted.
